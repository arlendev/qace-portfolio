# QA Projects

## QAce Bug Lab

### Overview

QAce Bug Lab is a custom-built testing laboratory created to support my Quality Assurance learning journey.

The project was designed to provide a controlled environment where I can practice software testing techniques, create documentation, execute test cases and simulate real QA workflows.

Rather than using third-party applications, I chose to build my own testing environment so I could fully understand both the system behavior and the testing process.

---

### Project Objectives

* Practice manual testing activities
* Create and maintain test documentation
* Develop structured test cases
* Execute regression testing
* Validate bug fixes
* Simulate real QA workflows
* Support future test automation initiatives

---

### Current Features

* User authentication
* Bug catalog
* Bug details modal
* Bug creation
* Bug update
* Bug deletion
* Search functionality
* Filtering functionality
* Local storage persistence
* Responsive user interface
