# QA Skills

## Manual Testing

Designing and executing functional test cases to validate application behavior, verify requirements and identify defects.

Practiced through hands-on testing activities, crowdtesting platforms and the QAce Bug Lab project.

---

## Test Case Design

Creating structured test cases with clear preconditions, test steps and expected results.

Applied through the development of test documentation for login, search, filter, CRUD operations and data persistence scenarios within QAce Bug Lab.

---

## Regression Testing

Building and maintaining regression test suites to verify that application changes do not introduce unintended defects.

Currently developing regression testing practices through the continuous evolution of QAce Bug Lab.

---

## Defect Reporting

Documenting defects with clear reproduction steps, expected behavior and observed results.

Focused on producing reports that facilitate communication between testers, developers and stakeholders.

---

## Exploratory Testing

Performing exploratory testing sessions to investigate system behavior, identify edge cases and discover unexpected issues.

Particularly interested in understanding how users interact with software and where failures may occur outside predefined scenarios.
